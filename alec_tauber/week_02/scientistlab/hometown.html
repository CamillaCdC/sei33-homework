Fun Facts/ Hometown

  1. James Werner Zawinski (born November 3, 1968), commonly known as jwz, is an American computer programmer
  2. He contributed free software projects such as Mozilla and XEmacs, and early versions of the Netscape Navigator web browser.
  3. He maintains the XScreenSaver project which provides screenblanking for Mac OS X and for Unix and Unix-like computer operating systems using the X Window System.
